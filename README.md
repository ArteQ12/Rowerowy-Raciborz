# [Rowerowy 🚴 Racibórz](https://rowerowy-raciborz.netlify.app/)

Strona internetowa zbierająca dane o ścieżkach rowerowych na terenie miasta Racibórz.

Projekt umożliwia przeglądanie infrastruktury rowerowej miasta w formie **interaktywnej mapy**, wraz z dodatkowymi materiałami i analizą jakości tras.

## Funkcje
- interaktywna mapa ścieżek rowerowych
- nagrania z przejazdów wybranych tras
- analiza i ocena jakości każdej ścieżki

## Zastosowane technologie
- HTML
- CSS
- JS *(od 2025)*
- Google Maps
- Fontello
- Haikei

## Historia projektu

- **Maj 2024** – projekt stworzony jako praca zaliczeniowa na zajęcia w technikum  
- **Listopad 2025** – wprowadzone poprawki i dostosowanie strony do urządzeń mobilnych
